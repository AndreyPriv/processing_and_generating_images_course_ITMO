Предмет: Обработка и генерация изображений

Задача: Множественная классификация объектов (10 классов)

Классы:

- airplane — самолеты
- automobile — автомобили
- bird — птицы
- cat — кошки
- deer — олени
- dog — собаки
- frog — лягушки
- horse — лошади
- ship — корабли
- truck — грузовики
  
Датасет: CIFAR-10

Архитектура: ResNet-18

Гиперпараметры:
- batch size: 32
- optimizer: Adam
- learning rate: 0.01
- loss: cross-entropy
- device: cuda
- epochs: 10
